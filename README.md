# Project Overview

This project demonstrates a Security Operations Center (SOC) monitoring workflow using Wazuh SIEM.

The lab focuses on monitoring endpoint activity, collecting security logs, analyzing alerts, investigating suspicious events, identifying Indicators of Compromise (IOCs), and documenting incident response activities.

---

# Objectives

- Configure Wazuh SIEM
- Monitor endpoint security events
- Analyze authentication logs
- Investigate security alerts
- Perform IOC analysis
- Correlate security events
- Practice incident investigation
- Document SOC workflows
